## 📖 项目简介 | Project Introduction

**JLU Toolkit（吉大工具箱）** 是一个专为吉林大学在校学生设计的轻量级 Web 工具集合。项目采用纯前端技术构建，无需后端部署，旨在通过一系列小而美的功能模块，提升同学们的学习效率与校园生活体验。

### ✨ 核心特色 | Key Features

- 🎨 **7种主题随心切换**：支持紫/红/蓝/绿/黄/黑白/科幻七种配色方案，适配不同使用场景与审美偏好
- 🌐 **中英双语支持**：完整支持中文与英文界面切换，方便国际学生使用
- 📱 **响应式设计**：完美适配桌面端与移动端，随时随地想用就用
- ⚡ **纯前端架构**：所有数据本地存储（LocalStorage），无需登录，即开即用，保护隐私
- 🧩 **模块化架构**：工具间相互独立，可单独使用，也可通过首页统一导航
